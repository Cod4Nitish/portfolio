## About The Project

Drawing inspiration from award-winning sites like [Jackie Hu](https://www.landing.love/sites/jackiehu/) and Framer's top workshops, this portfolio redefines standard design. A perfect blend of monochromatic elegance, glassmorphism, floating 3D geometry, and smooth scroll animations. The layout is engineered to be versatile, making it extremely easy to plug in your own data while looking breathtakingly premium.

## Tech Stack

This project is built with the most modern and performant technologies in the React ecosystem:

| Role | Technology |
| :--- | :--- |
| **Framework** | React 19 + Vite |
| **3D Engine** | Three.js + React Three Fiber + Drei |
| **Animations** | Framer Motion |
| **Styling** | Vanilla CSS (Premium Glassmorphic System) |
| **Icons** | Lucide React |

## Features

- 🌌 **Interactive 3D Canvas:** A dynamic, floating icosahedron and distorting sphere built with R3F that responds to user intent.
- ✨ **Framer Motion Choreography:** Every entrance, exit, and hover state is physically backed via spring animations.
- 🎨 **Deep Dark Glassmorphic Theme:** Inspired by modern dark mode UX patterns, using `backdrop-filter` and curated neon accents (`violet` and `cyan`).
- 📱 **Fully Responsive:** Custom CSS grids ensure optimal viewing from mobile to 4K displays.
- 📬 **Live Contact Flow:** Pre-configured Web3Forms integration out of the box.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and NPM installed.
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Cod4Nitish/portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
4. Build for production
   ```sh
   npm run build
   ```
